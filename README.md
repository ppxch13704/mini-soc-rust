# 🛡️ SOC Rust Projects

This repository contains a collection of Security Operations Center (SOC) tools built with Rust. It is divided into two main projects, evolving from a basic log analyzer to a fully interactive CLI dashboard.

## 📂 Repository Structure

* **`mini-soc-rust/`**: The initial, lightweight version focused on basic log parsing and threat detection.
* **`soc-dashboard/`**: The advanced version featuring an interactive Command Line Interface (CLI), colored output, and report generation capabilities.

---

## How to run 
After enter the directory using "cd <name-of-directory>"
then run using "cargo run" or "cargo run --bin <name-of-project>"
